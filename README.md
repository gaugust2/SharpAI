# SharpAI

## .env file variables

OPENAI_API_KEY=""
DB_USER=""
DB_PASSWORD=""
SPORTS_DATA_API_KEY=""

## To run this project

Make sure your .env file has all necessary variables. Open a terminal and run the command:

'''npm install'''

Then run either of the following commands:

'''npm start'''
'''npm run dev'''

The backend will then be accessible via localhost:3000.
